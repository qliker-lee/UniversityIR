
# University Management Strategy Analysis Platform 설명서

## 개요

본 프로그램은 대학 총장 및 기획처 관점에서 대학의 경쟁력, 재무, 학생성과, 장학정책 등을 종합적으로 분석할 수 있도록 설계된 Streamlit 기반 Executive Analytics Platform입니다.

단순 통계 조회가 아니라 데이터 기반 전략 의사결정을 지원하는 것을 목표로 합니다.

---

# 1. Executive Summary Dashboard

총장이 가장 먼저 확인하는 메인 대시보드입니다.

## 제공 기능

- 전체 대학 수
- 평균 등록금
- 평균 장학금 수혜율
- 평균 취업률
- 전체 학생 수
- 주요 KPI 카드 시각화

## 특징

- KPI Metric Card 제공
- Plotly 기반 인터랙티브 시각화
- 대학 평균 대비 현재 위치 즉시 확인 가능

## 활용 예시

- 우리 대학 등록금 수준은 전국 대비 어떤가?
- 장학금 정책은 경쟁력 있는가?
- 취업성과는 평균 이상인가?

---

# 2. 대학 경쟁력 분석

## 2-1. 취업률 vs 등록금 분석

산점도 기반 분석 기능을 제공합니다.

### 분석 항목

- X축: 등록금
- Y축: 취업률
- 점 크기: 학생 수
- 점 색상: 대학 유형 또는 지역

### 분석 가능 내용

- 등록금이 높지만 취업성과가 낮은 대학 탐지
- 저비용 고효율 대학 탐지
- 경쟁대학 포지셔닝 비교

---

## 2-2. 장학금 효과 분석

### 분석 항목

- 장학금 수혜율
- 학비감면율
- 취업성과 상관관계

### 활용 목적

- 장학정책 효과 검증
- 장학금 대비 실제 성과 분석
- 재정 효율성 판단

---

# 3. 통계 기반 경영 분석

---

## 3-1. Correlation Analysis (상관분석)

Pearson Correlation 기반 분석 수행

### 분석 대상 예시

- 등록금 ↔ 취업률
- 장학금 ↔ 취업률
- 학생 수 ↔ 취업률
- 학비감면 ↔ 재학생 규모

### 시각화

- Correlation Heatmap
- 양/음 상관 강도 표시

### 활용 목적

- 실제 성과에 영향을 주는 요인 발견
- 정책 우선순위 도출

---

## 3-2. Regression Analysis (회귀분석)

Linear Regression 기반 예측 분석 기능을 제공합니다.

### 예시 모델

```python
취업률 = a + b*(등록금) + c*(장학금)
```

### 제공 결과

- 회귀계수
- 결정계수(R²)
- 변수 영향도 분석

### 총장 관점 활용

- 무엇이 취업률을 움직이는가?
- 등록금 정책이 실제 경쟁력 향상에 도움이 되는가?

---

# 4. 대학 군집분석 (K-Means Clustering)

대학을 자동으로 유형화하여 경쟁 포지션을 분석합니다.

## 군집 예시

- 상위 경쟁 대학군
- 고비용 저성과 대학군
- 저비용 고효율 대학군
- 위험군 대학

## 분석 기준

- 등록금
- 장학금
- 취업률
- 학생 규모

## 시각화

- Cluster Scatter Plot
- 군집별 평균 KPI

## 활용 목적

- 경쟁대학 벤치마킹
- 구조개혁 대상 탐지
- 대학 포지셔닝 전략 수립

# 8. 전략 의사결정 지원 기능

### 사례 1

장학금 확대가 취업성과 향상으로 연결되지 않음

→ 장학 정책 재검토 필요

→ 교육 품질 개선 필요

---

### 사례 3

학생 규모는 크지만 성과가 낮은 군집

→ 구조개혁 우선 검토 대상
